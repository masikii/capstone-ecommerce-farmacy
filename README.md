#Hospital Pharmacy E-Commerce System
Capstone Project – Module 1 (Python Fundamental)

##Project Description
This project is a **Command Line Interface (CLI)** based mini application developed using Python.  
It is designed to manage hospital pharmacy operations efficiently.

The system manages:
- Medicine Data
- Customer Data
- Sales Transactions
- Sales Reports

This project applies fundamental Python concepts such as:
- Data Collection (List & Dictionary)
- Conditional Statements
- Looping
- Regular Functions
- Input Validation
- System Integration

---

##Background
Hospital pharmacy management is often handled manually, which can lead to:
- Stock miscalculations
- Transaction errors
- Poor data tracking

Therefore, this system was developed to:
- Reduce human error
- Automate transaction recording
- Manage medicine stock systematically
- Generate automatic sales reports

---

##Main Features

###Medicine Management (CRUD)
- Add medicine
- View medicine list
- Update medicine data
- Delete medicine

###Customer Management (CRUD)
- Add customer
- View customer data

###Sales Transaction
- Customer & medicine ID validation
- Automatic stock update
- Automatic discount system:
  - 10% discount for total purchase > 100,000
  - 5% discount for quantity ≥ 10
- Transaction data recording

###Sales Report
- Total revenue
- Best-selling medicine
- Most active customer
